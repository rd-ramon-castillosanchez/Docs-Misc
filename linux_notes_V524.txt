How to change the Date
======================

 date 033121422003.55
Mon Mar 31 21:42:55 PST 2003

The above command does not use the -s option, and the fields are arranged like this:  MMDDhhmmCCYY.ss
where MM = month, DD = day, hh = hour, mm = minute, CCYY = 4 digit year, and ss = seconds.

<vi>
vi
==

move cursor
============

h === lef
l === right
j === down
k === up
w === a word forward
b === a word backward


a === append to the end of the word
o === open a newline below the cursor
r === replace a character
x === delete 1 charater where the cursor is on
5x === delete 5 charaters from where the cursor is on
dw === delete a word
dd === deletes a line
4dd === deletes 4 lines

u === undo last command (ex. search and replace)

Copy and Paste is Yanking and Putting
=====================================

shift y === Yank
shift p === Put
shift j === Normal-mode command will join the current line and the next

Search and replace

/ === searchs forward (find a word)
? === searchs backward (find a word)
n === after searched jumps to next ocurrence 


shift g   ==== Bottom of the file
10 shift g ==== goes to line 10 of the file
ctrl + b (back on page at the time)
ctrl + n (next)
ctrl + f (one page forward at the time)
0 === begining of the line
$ === end of the line
: === Command mode
:1 [Enter]   Go to page 1
:set === shows all the vi enviroment variables and its settings
:set nu === sets the numbering on (numbers at the left of lines in the vi view)
:set ic === set ignore case on (bash and BASH are the same)
:set noic === goes back to case sensitve
:r !date === under command mode executes the linux command date and appends it to the open file




 :%s/search_string/replacement_string/g

 :3,5s/search_string/replacement_string/g === only between lines 3 to 5

 :.$s/search_string/replacement_string/g === only between cursor and the end of the file

 :%s/\n/ /   ==== substitutes *ALL* new lines in file , creating only one long line

 :%s/<ctrl>v<ctrl>m//g

:%s/^M//g      === CONTROL-M (Fix windows to Unix)

:%s/.$//      === any character (.) exactly at the end of the linf ($)
